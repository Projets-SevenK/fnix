# FNIX

Site vitrine-commerce pour le premier drop FNIX.

## Démarrage

```bash
npm install
npm run dev
```

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- Vercel

## Contexte

Lire `CLAUDE.md` avant de développer.
La maquette validée doit être placée dans `prototype/`.
Les tâches sont suivies dans `taches/a-faire.md`.
Les décisions et corrections importantes sont suivies dans `taches/lecons.md`.
